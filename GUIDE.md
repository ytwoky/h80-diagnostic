## Что было сделано

### Исправленные баги в коде
| Место | Проблема | Решение |
|-------|----------|---------|
| `render_di` | `cur = value or ref` заменял `0.0` на эталон | `cur = val if val is not None else ref` |
| `render_economics` | `npv == best` — сравнение float через `==` | `abs(npv - best) < 1.0` |
| `render_di` | Метка «DI аэро (ANSYS)» неточная | Переименовано в «DI (ΔN=0, ANSYS CFD)» |
| `render_report` | В формуле писалось `N_факт` вместо `N_привед` | Исправлено |
| Радар-чарт | Шкала без единиц | Добавлено `%` к меткам |

### Добавленные функции
- **Тулбар на графиках** — `NavigationToolbar2Tk` на всех трёх вкладках (Тренды, DI, Экономика). Даёт: зум, панорамирование, координаты курсора в реальном времени, сохранение картинки.
- **Таблица сценариев** — вместо одной длинной строки теперь нормальная таблица с колонками: Сценарий / Инвестиции / Ежегодно / NPV / Окупаемость.

### Настроен GitHub Actions CI
- Каждый `git push` → автоматическая сборка для **macOS** (arm64) и **Windows** (x64).
- Артефакты хранятся 30 дней во вкладке Actions.
- Тег `v*` → создаётся публичный **Release** со скачиваемыми ZIP-архивами.

## Диагностический индекс DI

### Формула
```
DI = 0.60 × DI_power + 0.40 × DI_aero

DI_power = min(max(ΔN, 0) / 10%, 1) × 100
DI_aero  = Σ weight_i × min(|cur_i - ref_i| / range_i, 1) × 100
```

### Веса аэродинамических параметров
| Параметр | Вес |
|----------|-----|
| P7 — коэф. потерь ζ | 0.50 |
| P1 — массовый расход G | 0.30 |
| P8 — число Маха | 0.15 |
| P9 — угол потока α | 0.05 |

### Пороги статусов
| DI | ΔN | Ra | Статус |
|----|----|----|--------|
| < 30% | < 2% | < 7.5 мкм | НОРМА |
| ≥ 30% | > 5% | > 50 мкм | ПРЕДУПРЕЖДЕНИЕ |
| ≥ 70% | > 10% | > 85 мкм | КРИТИЧНО |

### Знак ΔN
```
ΔN = (N_ожид − N_привед) / N_ожид × 100%
+ → дефицит мощности (деградация)
− → мощность выше модели (проверь N_ном)
```

## Экономический модуль

### Фактор деградации (топливо)
```
ΔF = ΔN/100 + 0.3×Δζ/100 + 0.5×ΔG/100
Доп. расход = F_base × ΔF  [кг/ч]
Потери/год  = Доп. расход × налёт × цена_топлива
```

### Три сценария ТО
| Сценарий | Инвестиции | Ежегодные затраты |
|----------|-----------|-------------------|
| Status Quo | 0 | 100% топливных потерь |
| Промывка ПЧ | стоимость промывки | 2×промывки/год + 45% потерь |
| Полный ремонт MRO | стоимость MRO | 0 |

### NPV (метод из Гасленко, МГТУ ГА)
```
NPV_j = −K_0 − Σ(C_год / (1+r)^t),  t=1..j
Лучший сценарий → наименьшие суммарные затраты (NPV ближе к 0)
```
Совпадает с методикой из учебника «Экономика предприятий», Практическое занятие №9.
